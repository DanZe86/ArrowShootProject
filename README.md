# ArrowShootProject 

## Opis projektu (PL)

ArrowShootProject jest projektem edukacyjnym polegającym na symulacji zjawiska fizycznego związanego
z ruchem pocisku.

Projekt przedstawia uproszczoną symulację mechaniki strzału z łuku, uwzględniając podstawowe pojęcia
fizyczne, takie jak kierunek ruchu, przebyta odległość oraz logika poruszania się obiektu w przestrzeni.

Implementacja została wykonana w języku Python i ma na celu zaprezentowanie, w jaki sposób wybrane
zjawiska fizyczne mogą zostać odwzorowane w postaci algorytmów oraz kodu źródłowego.

Główne cele projektu:
- symulacja wybranego zjawiska fizycznego przy użyciu języka Python
- uwzględnienie podstawowych zasad fizyki w implementacji symulacji
- zastosowanie podstawowych zasad programowania
- stworzenie czytelnej i uporządkowanej struktury kodu

Projekt ma charakter edukacyjny i został przygotowany jako część nauki programowania oraz modelowania
zjawisk fizycznych.

## Project Description (EN)

ArrowShootProject is an educational project focused on simulating a physical phenomenon related to
projectile motion.

The project presents a simplified simulation of archery mechanics, taking into account fundamental
physical concepts such as movement direction, distance traveled, and object motion logic in space.

The implementation is written in Python and aims to demonstrate how selected physical phenomena can
be represented using algorithms and source code.

Main project objectives:
- simulation of a selected physical phenomenon using Python
- application of basic physics principles in the simulation implementation
- application of fundamental programming concepts
- creation of a clear and well-structured codebase

This repository serves as an educational example of combining physics simulation with software
development practices.
